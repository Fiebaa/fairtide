How It Works

Three inputs, one fair price. Integrate with a single API call.

1

Set Your Base Price

You decide the standard price for your product — a coffee, a meal, a ticket. That's your anchor.

2

Customer Shares Income

At checkout, the customer selects their income range. No verification needed — it's trust-based.

3

API Returns Fair Price

Fairify calculates an adjusted price using income and location factors. Lower income pays less, higher income pays a bit more.

Revenue-Neutral by Design

The question every business owner asks: "Won't I lose money?"

Fairify tracks every transaction's delta — the difference between your base price and the fair price. When too many discounts accumulate, the algorithm automatically dampens future adjustments, nudging prices back toward your base price.

Over time, discounts given to lower-income customers are offset by premiums from higher-income customers. Your revenue stays balanced.

Balanced — Revenue on track
Recovering — Dampening active, nudging toward base
Surplus — Premiums exceeded discounts, relaxing factors

Simple API Integration

One POST request. That's the entire integration.

Request
POST /v1/calculate
Authorization: Bearer your-api-key
Content-Type: application/json

{
  "basePrice": 4.50,
  "annualIncome": 28000,
  "locationId": "berlin-de"
}
Response
{
  "fairPrice": 3.78,
  "breakdown": {
    "basePrice": 4.50,
    "incomeFactor": 0.85,
    "locationFactor": 1.05,
    "adjustedIncomeFactor": 0.80,
    "fairPrice": 3.78
  },
  "balanceStatus": "balanced"
}

Get Started

1

Create a Realm

Each business gets its own realm with an API key and isolated balance tracking.

POST /v1/realms
{ "name": "My Cafe" }
2

Use Your API Key

Include the returned API key in every request as a Bearer token.

3

Check Your Balance

Monitor your realm's revenue balance anytime.

GET /v1/realms/:id/balance
